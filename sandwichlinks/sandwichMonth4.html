Week 1

Adding Functionality to Web Application ++ Synchrotron Tour

This week, I was mainly working on adding a functionality to my team's latest Software (The Scheduler) - I was specifically adding a "ManageTeams" page that will be used to manage the teams that will be using the Scheduler application.
In implementing this, I wrote back-end (Java) code that includes web methods (SOAP) - this created the opportunity for me to learn more about web methods and gatekeepers. I also had to interact with the database from the Java code, so I was able to learn more in regards to JPA (Java Persistence API) - Using Entity Manager, persisting data, etc.
I also wrote front-end code in order to display results and handle the "team management" of the application. I used ASP.NET and Telerik tags - which provided me with the opportunity to learn more about interacting with telerik tags such as "RadListBox", and ASP.NET tags such as "DropDownList".
To start this, I initially had to draw a mockup of what the page will look like. And was able to complete this task at the end of the week.

Week 3

Integration Test ++ Sharepoint 2013 on Virtual Box

I was able to learn about Integration Test this week. I wrote a few Integration tests for the web server methods I had written previously while adding the "Teams management" functionality

Sharepoint 2013
I was able to take a significant amount of time setting up and installing Sharepoint 2013 on a virtual machine. Week 4

Released Software To Production [Jenkins] ++ Custom Page layout On Sharepoint 2013 ++ SOTW

I was able to release my team's main application to production for the first time. This required extra checking of things (e.g Database scripts that runs running, etc) and also required extra caution.
I carried out the deployment of the application using Jenkins (our continuous Integration System).

Custom Page layout On Sharepoint 2013
After installing Sharepoint 2013, my task was then to read tutorials and basically have a good go at how customising a site, web-pages, master pages, layout pages, and making them all fit into each other - hence designing them appropriately on Sharepoint 2013 - that was initially not easy for me to get my head around, but then it became clear and fun!

Supporter Of The Week (SOTW)
It was my turn to support the users of our application. Not a very busy week, but it still involved me having to show quick problem solving skills in order to resolve one issue or the other. I also had to write a couple of fancy SQL scripts.
