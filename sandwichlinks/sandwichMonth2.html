Week 1

Acceptance Testing - BLAATs

This week, I got introduced to a way of using code to test code at an even higher level using a framework put together by a colleague at work. The framework (called BLAATs) uses Selenium and this enables a programmer to be able to test applications on a web browser without the need to touch the mouse or keyboard - as the operations to be tested are coded and hence performed automatically. This framework includes the use of Command Line design pattern and Unit testing.
Pretty cool stuff
I was also "Supporter of the week" this week. So I get to communicate with users office on phone or via emails in order to support them with queries that may arise regarding new and existing tools and applications. I got to remotely log into a user's computer in order to rectify some issues with her copy of an application.
I wrote more unit tests for some methods in our application (using JMockIt)
